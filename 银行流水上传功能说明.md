# 银行流水上传功能说明

## 功能概述

银行流水上传功能允许用户上传银行流水压缩包文件，系统会自动解析并导入到案件数据库中。

## 前端页面

### 访问路径
- 路由：`/system/bank-statements`
- 菜单：侧边栏 "银行流水上传"

### 主要功能

1. **案件选择**
   - 下拉选择要上传流水的案件
   - 只显示用户有权限访问的案件

2. **文件上传**
   - 支持 ZIP/RAR/7Z 格式的压缩包
   - 支持同时上传多个文件
   - 支持拖拽上传
   - 需要对案件有写入权限（write 或 admin）

3. **任务列表**
   - 显示所有上传任务
   - 实时显示任务状态和进度
   - 支持按状态筛选（等待中/处理中/已完成/失败/已取消）

4. **任务管理**
   - 查看任务详情
   - 取消正在处理的任务
   - 删除任务记录（需要 admin 权限）

5. **统计信息**
   - 总任务数
   - 总文件数
   - 总记录数
   - 成功记录数

6. **自动轮询**
   - 当有任务在处理时，每3秒自动刷新任务列表
   - 任务完成后自动停止轮询

## 权限要求

- **上传文件**：需要对案件有 write 或 admin 权限
- **查看任务**：需要对案件有 read 权限
- **取消任务**：需要对案件有 write 权限
- **删除任务**：需要对案件有 admin 权限

## 使用流程

1. 登录系统
2. 点击侧边栏 "银行流水上传"
3. 选择要上传的案件
4. 点击 "上传流水文件" 按钮
5. 在对话框中选择案件和文件
6. 点击 "开始上传"
7. 等待任务处理完成
8. 查看任务详情和统计信息

## 技术实现

### 前端文件
- `/frontend/src/views/System/BankStatement/index.vue` - 主页面组件
- `/frontend/src/api/bankStatement.js` - API 接口封装

### 后端接口
- `POST /api/v1/cases/{case_id}/bank-statements/upload` - 上传文件
- `GET /api/v1/cases/{case_id}/bank-statements/tasks/{task_id}` - 查询任务进度
- `GET /api/v1/cases/{case_id}/bank-statements/tasks` - 查询任务列表
- `POST /api/v1/cases/{case_id}/bank-statements/tasks/{task_id}/cancel` - 取消任务
- `DELETE /api/v1/cases/{case_id}/bank-statements/tasks/{task_id}` - 删除任务
- `GET /api/v1/cases/{case_id}/bank-statements/statistics` - 获取统计信息

## 注意事项

1. 上传的文件必须是压缩包格式（ZIP/RAR/7Z）
2. 文件上传超时时间为5分钟
3. 任务处理是异步的，需要等待 Celery 任务完成
4. 确保 Celery 服务已启动
5. 确保 Redis 服务已启动（Celery 依赖）
