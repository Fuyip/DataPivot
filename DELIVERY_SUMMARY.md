## 🚀 快速开始指南

### 方式1: 本地开发
```bash
# 1. 初始化项目
bash init.sh

# 2. 激活虚拟环境
source venv/bin/activate

# 3. 配置数据库
vim .env

# 4. 测试现有功能
python 导入云搜人员基本信息.py
```

### 方式2: Docker部署
```bash
# 1. 配置环境变量
cp .env.example .env
vim .env

# 2. 启动所有服务
docker-compose up -d

# 3. 查看服务状态
docker-compose ps

# 4. 查看日志
docker-compose logs -f
```

## ⚠️ 重要提示

### 数据安全
- 🔒 **内网部署**: 必须部署在内网环境，不得暴露到公网
- 🔑 **强密码**: 生产环境使用强密码，定期更换
- 💾 **数据备份**: 每天自动备份数据库，保留30天
- 📝 **操作日志**: 记录所有操作日志，定期审查
- 🛡️ **权限控制**: 实施严格的权限管理，最小权限原则

### 技术注意事项
- 🗄️ **数据库**: 现有数据库通过database.py连接，请勿随意修改
- 📦 **依赖管理**: 使用requirements.txt管理Python依赖
- 🐳 **容器化**: 推荐使用Docker部署，环境一致性好
- 📊 **性能**: 大文件处理需要优化，建议使用异步任务
