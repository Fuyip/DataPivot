1. **更新API调用** (`frontend/src/api/importRule.js`)

**使用方式**：

1. 打开字段映射配置对话框
2. 选择数据类型标签页
3. 点击"添加字段"
4. 在"数据库字段"列：

   - 可以搜索字段名或标签
   - 选择一个字段，或输入自定义字段名
5. 填写CSV列名和其他配置
6. 保存

**优势**：
- ✅ 减少输入错误
- ✅ 提供字段说明，更易理解

- ✅ 仍支持自定义字段名（灵活性）

## 关于"新建模板不显示"的问题

如果新建模板后列表没有自动刷新，请检查：

### 可能的原因

1. **浏览器缓存**
   - 按 Ctrl+Shift+R (Windows) 或 Cmd+Shift+R (Mac) 强制刷新
   - 清除浏览器缓存

2. **前端状态未更新**
   - 检查浏览器控制台是否有JavaScript错误
   - 检查Network标签，确认API请求成功返回

3. **搜索条件过滤**
   - 检查是否设置了搜索条件
   - 点击"重置"按钮清除筛选

### 调试步骤

1. 打开浏览器开发者工具 (F12)
2. 切换到 Console 标签
3. 创建新模板
4. 查看是否有错误信息
5. 切换到 Network 标签
6. 查看 API 请求：
   - POST `/api/v1/import-rules/templates` - 创建模板
   - GET `/api/v1/import-rules/templates` - 获取列表
7. 确认两个请求都返回 200 状态码

### 临时解决方案

如果自动刷新失败，可以：
1. 手动点击"查询"按钮刷新列表
2. 或刷新整个页面 (F5)

## 测试建议

### 完整流程测试

1. **创建新模板**
   - 访问"系统管理" → "导入规则管理"
   - 点击"新增模板"
   - 填写模板名称和描述
   - 保存并确认列表中显示新模板

2. **配置字段映射**
   - 点击新模板的"字段映射"按钮
   - 选择"交易明细"标签页
   - 点击"添加字段"
   - 在数据库字段下拉框中选择"card_no (交易卡号)"
   - 填写CSV列名，如"交易卡号"
   - 选择字段类型为"卡号"
   - 继续添加其他必需字段
   - 保存配置

3. **配置清洗规则**
   - 点击"清洗规则"按钮
   - 添加通用清洗规则
   - 正则表达式：`\t| |"|\\|(nan)|\\|,`
   - 保存

4. **使用模板导入数据**
   - 访问"银行流水上传"页面
   - 选择案件
   - 点击"上传流水文件"
   - 在导入规则下拉框选择刚创建的模板
   - 上传测试文件
   - 验证数据导入结果
