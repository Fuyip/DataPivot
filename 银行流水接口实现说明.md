## API端点

### 上传文件
```
POST /api/v1/cases/{case_id}/bank-statements/upload
```
- 上传银行流水压缩包
- 支持多文件上传
- 需要write权限

### 查询任务进度
```
GET /api/v1/cases/{case_id}/bank-statements/tasks/{task_id}
```
- 查询任务处理进度
- 需要read权限

### 查询任务列表
```
GET /api/v1/cases/{case_id}/bank-statements/tasks
```
- 查询案件的所有任务
- 支持分页和状态筛选

### 取消任务
```
POST /api/v1/cases/{case_id}/bank-statements/tasks/{task_id}/cancel
```
- 取消正在处理的任务
- 需要write权限

### 删除任务
```
DELETE /api/v1/cases/{case_id}/bank-statements/tasks/{task_id}
```
- 删除任务记录和文件
- 需要admin权限

### 获取统计
```
GET /api/v1/cases/{case_id}/bank-statements/statistics
```
- 获取案件的银行流水导入统计

### 3. 启动Redis服务
```bash

```

### 4. 启动Celery Worker
```bash

```

### 5. 启动FastAPI应用

