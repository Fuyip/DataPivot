## 🎯 技术栈总结

### 前端技术栈
- **框架**: Vue 3.4+
- **UI组件库**: Element Plus 2.5+
- **状态管理**: Pinia 2.1+
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **构建工具**: Vite 5.0+
- **图表库**: ECharts 5

### 后端技术栈
- **框架**: FastAPI 0.110+
- **ORM**: SQLAlchemy 2.0+
- **数据处理**: Pandas 2.2+ / NumPy 1.26+
- **任务队列**: Celery 5.3+ (可选)
- **缓存**: Redis 6.0+ (可选)
- **日志**: Loguru 0.7+
- **文档解析**: BeautifulSoup4, openpyxl

### 数据库
- **主数据库**: MySQL 8.0+
- **字符集**: utf8mb4

### 部署
- **容器化**: Docker + Docker Compose
- **Web服务器**: Nginx 1.18+
- **ASGI服务器**: Gunicorn + Uvicorn
