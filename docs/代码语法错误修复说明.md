# 代码语法错误修复说明

## 修复日期
2026-03-09

## 问题描述

在修复查询功能兼容性时，`get_case_card()` 方法中出现了重复的 WHERE 子句，导致语法错误。

**错误代码位置**：`backend/services/case_card_service.py` 第131-132行

**错误内容**：
```python
else:
    query_sql = """
        SELECT id, case_no, source, user_id, card_no, bank_name, card_type,
               add_date, batch, is_in_bg, is_main
        FROM case_card
        WHERE id = :card_id
    """
    WHERE id = :card_id  # 重复的行
"""
```

## 问题原因

在添加动态字段检查逻辑时，复制粘贴代码时产生了重复的 WHERE 子句。

---

## 解决方案

删除重复的 WHERE 子句，保持正确的 SQL 语句结构。

**修改位置**：`backend/services/case_card_service.py` 第124-134行

**修改后的代码**：
```python
else:
    query_sql = """
        SELECT id, case_no, source, user_id, card_no, bank_name, card_type,
               add_date, batch, is_in_bg, is_main
        FROM case_card
        WHERE id = :card_id
    """

result = db.execute(text(query_sql), {"card_id": card_id}).fetchone()
```
