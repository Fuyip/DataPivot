## 📋 项目目标

为 DataPivot 案件银行卡管理系统实现完整的导入任务跟踪功能，包括：

1. ✅ 导入历史记录管理
2. ✅ 银行名称智能匹配
3. ✅ 批量删除导入数据
4. ✅ 一键重新匹配银行

---

## 🎯 核心功能

### 1. 导入任务跟踪
- 每次导入创建任务记录
- 记录文件名、统计信息、错误详情
- 支持查看历史导入任务

### 2. 银行名称智能匹配
- 基于 BIN 码自动匹配银行
- 匹配失败不拒绝导入（设为 NULL）
- 支持一键批量重新匹配

### 3. 批量任务管理
- 查看所有导入任务
- 查看错误详情
- 根据任务ID批量删除数据

## ⚠️ 注意事项

### 数据库迁移
- 只为 test01_GXMLM 数据库添加了字段
- 其他案件数据库需要手动执行迁移脚本

- 无法追溯历史导入任务

### 权限要求
- 删除任务数据需要 admin 权限
- 一键匹配银行需要 write 权限
