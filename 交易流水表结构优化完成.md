## 修改范围

## 优化效果
1. **存储空间优化**：字段长度更加合理，减少不必要的存储空间浪费
2. **查询性能提升**：较短的字段长度可以提高索引效率和查询速度
3. **数据类型优化**：`rival_trade_balance` 从字符串改为 double 类型，更适合数值计算
4. **标准化统一**：所有交易流水相关表结构完全一致，便于维护

## 注意事项
1. 此修改仅更新了 SQL 模板文件，不影响现有数据库
2. 新建案件数据库时会自动使用新的表结构
3. 如需更新现有数据库，需要执行 ALTER TABLE 语句
4. 建议在更新现有数据库前先备份数据

## 后续建议
如需将现有数据库更新为新结构，可以生成对应的 ALTER TABLE 迁移脚本。
