## ⚠️ 重要提示

### 关于数据库连接
运行 `verify_database.py` 时可能遇到连接被拒绝的错误，这是**正常的数据库安全配置**，不影响系统功能。

**解决方案**：
- 跳过数据库验证脚本
- 直接使用前端界面测试
- 或通过 API 测试脚本测试
- 详见：[数据库连接问题说明](./数据库连接问题说明.md)
